MarkDown写作

new

  • 新建一篇文章，会在 \source\_posts 目录下生成 title.md

    hexo new "title"

  • 新建页面，会在 \source\page 目录下生成 index.md

    hexo new page "page"

  • 发表草稿，不会在显示页面

    hexo new tra "caogao"

  • 将草稿发布

    hexo public "chage caogao to title"
